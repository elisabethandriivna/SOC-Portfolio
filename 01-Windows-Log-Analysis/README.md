# Windows Log Analysis

## Objective

Investigate Windows Security logs to identify user authentication events and detect suspicious activity.

---

## Scenario

A user reported unusual behavior on their Windows workstation.

The SOC team was asked to review Windows Event Logs to determine whether unauthorized access had occurred.

---

## Tools

- Windows Event Viewer
- Windows Security Logs
